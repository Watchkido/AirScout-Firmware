# Autonomous Sailing Arduino Project

Dieses Projekt steuert ein Mini-Windsurf-Roboter mit Arduino Mega.

## Verwendete Komponenten
- Arduino Mega 2560
- GPS-Modul
- Kompass
- Windsensor
- 2x Servomotoren
